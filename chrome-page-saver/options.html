sPageSaver Settings

Configure how articles are saved to your Downloads folder

Save Path

The path where articles will be saved. Create custom folder structures.
Available variables:
{domain} - Website domain
{year} - Year (e.g., 2024)
{month} - Month (01-12)
{day} - Day (01-31)
{date} - Full date (YYYY-MM-DD)
Example:
articles/

Filename Template

The naming pattern for saved HTML files.
Available variables:
{title} - Page title
{domain} - Website domain
{date} - Save date (YYYY-MM-DD)
{datetime} - Save datetime (ISO format)
Example output:
My-Article-Title.html
